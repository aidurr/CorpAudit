# Changelog

All notable changes to CorpAudit will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of CorpAudit
- Telemetry detection and reporting
- Bloat detection and analysis
- Permission auditing
- Fix generation and application
- Multiple output formats (text, JSON, Markdown)
- Configurable thresholds and patterns
- Safe mode for non-destructive operations
- Comprehensive documentation

### Features
- Scan for telemetry and data collection
- Detect resource-heavy applications
- Audit application permissions
- Generate fix scripts
- Apply fixes with rollback support
- Customizable configuration
- Cross-platform support (Linux)

### Security
- No telemetry or data collection
- Local-only operation
- Safe-by-default design

- First public release of CorpAudit
